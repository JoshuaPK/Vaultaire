--- layout: bootstrap title: Vaultaire - Document Scanning & Tagging ---

What is it?

Vaultaire is a graphical front-end to standard Linux command line utilities (scanimage, find, grep, etc.) that simplifies scanning, tagging, organizing, and retrieving of documents, letters, receipts, etc.

What does it do?

  • Scanning and importing documents into a simple folder scheme
  • Storing of user-defined meta data (date, category, title, tags) for each document
  • Optical character recognition of scanned documents
  • Advanced searching of document library

How do I get it?

  1. Clone from GitHub
  2. Build Vaultaire with make
  3. Install with make install
  4. Launch with /usr/bin/vaultaire
